Find The Word
===

[Download it from Google Play](https://play.google.com/store/apps/details?id=com.gamesrum.guesstheword)

How to build:
- npm i -g gulp
- npm install
  
  If you get an error related to bower, then do this:
  - npm i -g bower
  - bower install

- gulp --prod

Congrats, you've built the app. It is on `buil/` folder.
